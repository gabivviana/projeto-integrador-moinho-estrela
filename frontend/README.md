# Frontend - Controle de Divergência de Peso

Aplicação React simples para controle de divergência de peso.

## Instalação

```bash
npm install
```

## Execução

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`

## Funcionalidades

- Formulário para cadastro de notas fiscais
- Histórico de todas as notas cadastradas
- Ranking de fornecedores por divergência

## Tecnologias

- React 18
- Vite
- Axios
- CSS básico
